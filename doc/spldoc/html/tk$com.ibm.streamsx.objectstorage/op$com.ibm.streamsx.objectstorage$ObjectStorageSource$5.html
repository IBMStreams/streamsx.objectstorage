For IAM authentication the following authentication parameters can be used, but it is recommended to use the credentials parameter instead:
  • IAMApiKey

  • IAMServiceInstanceId

  • IAMTokenEndpoint - IAM token endpoint. The default is https://iam.bluemix.net/oidc/token.
