Operators
  • ObjectSink: The operator writes a object (create or replace object) to the Object Storage.

Composites

composite ObjectSink(output stream<$outStreamType> Out0; input In0)

The operator writes a object (create or replace object) to the Object Storage.

Parameters

  • objectStorageUserID: Object Storage user-id (Bluemix Object Storage credential)
  • objectStoragePassword: Object Storage password (Bluemix Object Storage credential)
  • objectStorageProjectID: Object Storage project-id (Bluemix Object Storage credential)
  • objectStorageContainer: Object Storage container (Bluemix Object Storage credential)
  • objectStorageAccessPoint: (Bluemix Object Storage credential) The Object Storage service provides different endpoints for interacting with the service. This operator uses the Identity V3 endpoint for authentication and uses Dallas as an access point. These endpoints can be updated to utilize different versions of the API or different access points as needed.
  • objectStorageAuthURL: This operator uses the Identity V3 endpoint for authentication.
  • preservePunctuation: If set to true then the operator forwards punctuation from input port 0 to output port 0. The default value is false.
  • objectName: The attribute of type rstring, which contains the object name.
  • objectData: The attribute of type rstring, which contains the object content.
  • outStreamType: The OutStream (output port 0) schema of this operator. All attributes matching with input port 0 attributes are forwarded.

Input Ports

  • In0: Tuples containing the objectName and objectData attributes. Window punctuation received on input port 0 is forwarded to output port 0, if the parameter preservePunctuation is set to true.

Output Ports

  • Out0: The output tuple contains the objectName. All matching attributes of the input tuple are forwarded to this output stream if present. Window punctuation received on input port 0 is forwarded, if the parameter preservePunctuation is set to true.
