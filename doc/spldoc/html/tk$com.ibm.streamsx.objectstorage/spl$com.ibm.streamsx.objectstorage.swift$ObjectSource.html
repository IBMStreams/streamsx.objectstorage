Operators
  • ObjectSource: The operator reads a object (get object content) from Object Storage.

Composites

composite ObjectSource(output Out0; input In0)

The operator reads a object (get object content) from Object Storage.

Parameters

  • objectStorageUserID: Object Storage user-id (Bluemix Object Storage credential)
  • objectStoragePassword: Object Storage password (Bluemix Object Storage credential)
  • objectStorageProjectID: Object Storage project-id (Bluemix Object Storage credential)
  • objectStorageContainer: Object Storage container (Bluemix Object Storage credential)
  • objectStorageAccessPoint: (Bluemix Object Storage credential) The Object Storage service provides different endpoints for interacting with the service. This operator uses the Identity V3 endpoint for authentication and uses Dallas as an access point. These endpoints can be updated to utilize different versions of the API or different access points as needed.
  • objectStorageAuthURL: This operator uses the Identity V3 endpoint for authentication.
  • preservePunctuation: If set to true then the operator forwards punctuation from input port 0 to output port 0. The default value is false and a window punctuation is generated on output port 0 after object is read.
  • objectName: The attribute of type rstring, which contains the object name.
  • objectData: The attribute of type rstring, which contains the object content.

Input Ports

  • In0: Tuples containing the objectName to read and objectData attribute to be filled. This operator forwards the input tuple and sets the two attributes of the input tuple specified with objectName and objectData parameter.

Output Ports

  • Out0: The schema of the output tuple is the same as the input tuple since it is the same tuple. This operator does not create a new output tuple, because it sets attributes of the input tuple and forwards this tuple. Window punctuation received on input port 0 is forwarded to output port 0, if the parameter preservePunctuation is set to true. Otherwise a window punctuation is generated after each input tuple.
