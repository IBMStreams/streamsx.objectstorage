S3Sink
The Java Operator S3Sink uses the S3 API interface to store files in IBM’s Cloud Object Storage System.
S3Source
The Java Operator S3Source uses the S3 API interface to read files from IBM’s Cloud Object Storage System.
SwiftSink
The operator writes a file (create or replace object) to the Object Storage.
SwiftSource
The operator reads a file (get object content) from Object Storage.
